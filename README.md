#  Sistema de Inventario en Python

##  Descripción
Sistema de inventario desarrollado en **Python** que permite gestionar productos desde consola.  
El proyecto simula un caso real de uso administrativo, implementando operaciones básicas como agregar, editar, eliminar y listar productos, con una lógica clara y estructurada.

Este proyecto fue creado con fines educativos y de práctica, enfocado en fortalecer **lógica de programación**, **estructuras de datos** y **organización del código**.

---

##  Funcionalidades

-  Agregar productos al inventario
-  Editar productos existentes
-  Eliminar productos
-  Listar todos los productos
-  Validación básica de entradas
-  Menú interactivo por consola

---

##  Conceptos aplicados

- Lógica de programación
- Estructuras de datos (listas y diccionarios)
- Funciones
- Condicionales y bucles
- Separación de responsabilidades
- Flujo de control

---

##  Tecnologías utilizadas

- **Lenguaje:** Python  
- **Entorno:** Consola  
- **Control de versiones:** Git / GitHub
